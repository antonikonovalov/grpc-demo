How this sample app works

  • Attendants are given a one-time use code for registering
  • Everytime a new name is entered with a code, they are sent to the server.
    • The code must have been given to the attendant, otherwise it will not work.
    • If the code has already been used, it will not work
  • After successfully using a code, the attendant count will be increased!
  • The names of the attendants will be listed on the dropdown element

Technologies Involved

  • AWS / Go - T2 Micro instance with Ubuntu, running a Go GRPC + HTTP server.
  • Raspberry Pi / Python - A GRPC Python client, with code that constantly polls the server.
  • Computer / Go - A GRPC Go client that generates a code for an attendant to register.
  • Website / JQuery / Bootstrap / Go Templates - This particular webpage :-)
